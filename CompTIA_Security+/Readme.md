## Beschreibung:
Die Zertifizierung "CompTIA Security+" richtet sich an IT-Fachleute, die grundlegende und praxisnahe Kompetenzen in der IT-Sicherheit nachweisen möchten. In diesem Kurs erwerben die Teilnehmenden fundiertes Wissen zu Sicherheitskonzepten, Bedrohungsanalysen, Absicherungs-mechanismen und Best Practices zur Absicherung von Netzwerken, Systemen und Daten. Die Zertifizierung bestätigt, dass Inhaber:innen über die notwendigen Fähigkeiten verfügen, Sicherheitsvorfälle zu erkennen, zu verhindern und darauf zu reagieren.

## Ziel der Prüfungsvorbereitung
Diese Prüfungsvorbereitung zielt darauf ab, Sie strukturiert und praxisnah auf die CompTIA Security+ Prüfung vorzubereiten. Sie erhalten die Prüfungsinhalte erklärt, üben typische Aufgabenstellungen und erhalten Tipps zur Prüfungsvorbereitung und zum Selbststudium, damit Sie die Prüfung mit Zuversicht antreten können.

## Hauptthemen:
- General security concepts
- Threats, attacks and vulnerabilities
- Security architecture and design
- Security operations

## Voraussetzungen:
- Grundkenntnisse in IT und Netzwerktechnik
- Grundverständnis von Betriebssystemen (Windows / Linux)
- Basiskenntnisse in Virtualisierung sind hilfreich

**Empfehlung:**  
Für praktische Übungen sollten Sie über ein Hostsystem verfügen, auf dem Sie virtuelle Maschinen installieren können. Falls gewünscht, bauen wir gemeinsam eine einfache Testumgebung auf. Gängige Virtualisierungssoftware wie VMware Workstation, VirtualBox oder Hyper-V ist geeignet. Typische virtuelle Maschinen für Übungen:
- Windows 10/11 (Client)
- Windows Server 2022 (Domain Controller, optional)
- Kali Linux (für Security-Tools, optional)
- Eine Linux-Distribution (z.B. Rocky Linux, optional)

> Hinweis: Stellen Sie sicher, dass Sie mindestens 8–16 GB Arbeitsspeicher und ausreichend Festplattenspeicher (mind. 50 GB) für die virtuellen Maschinen zur Verfügung haben.

## Zielgruppe:
Diese Prüfungsvorbereitung richtet sich an IT-Profis, System- und Netzwerkadministratoren, Entwickler:innen und alle, die eine fundierte Einstiegscertifizierung im Bereich IT-Security anstreben oder ihre Kenntnisse systematisch auffrischen möchten.
