## 📌 Überblick

Die **End-of-Month Tech Session** findet jeweils am **letzten Donnerstag eines Monats** von **13:00 bis 17:00 Uhr** statt.

Jede Session widmet sich einem **klar definierten Schwerpunktthema**. Ziel ist es, Wissen nicht nur zu vermitteln, sondern es direkt in einem praxisnahen Rahmen anzuwenden.

## ☁️ Session im Fokus: Microsoft Azure Grundlagen

In dieser Session erhalten die Teilnehmenden eine fundierte Einführung in die Grundlagen von **Microsoft Azure**.

### Inhalte
- Überblick über die **Azure-Architektur**
- Einführung in wichtige **Azure-Dienste**
- Verwaltung von **Azure-Ressourcen**
- Bereitstellung und Betrieb von **Cloud-Infrastruktur**

### Ziel
Die Teilnehmenden lernen, wie Azure-Dienste effizient eingesetzt und verwaltet werden können, um moderne IT-Infrastrukturen in der Cloud aufzubauen und zu betreiben.
